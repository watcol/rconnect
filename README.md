A [KDE Connect](https://kdeconnect.kde.org) server/client implementation in Rust.

## Usage
Add to your `Cargo.toml`:
```toml
[dependencies]
rconnect = "0.1.0"
```

API Documents are available on [docs.rs](https://docs.rs/rconnect).
